# Harden rhyme model cache: replace pickle with JSON
kind: debt
tags: rhyme, security
created: 2026-06-19T23:44Z

- 2026-06-19T23:45Z The phonetic/transcription model caches in ~/.slithyt/data are written with pickle.dump (cli.py build-cache) and read with pickle.load (rhyme.py load_phonetic_model/load_transcription_model). pickle.load on a file an attacker could plant is an RCE vector; the security hook flags it. Low risk today (built locally from the bundled CMU dict, no untrusted input), but switching to JSON would harden it: the phonetic model is dict[tuple->list[str]] (tuple keys need encoding) and the transcription model is dict[str->list[str]] (directly JSON-able).
